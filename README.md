# Monex

Modern currency conversion with real-time exchange rates.

Monex is a web application that provides fast, reliable, and accurate currency conversion with live exchange rates, historical trends, and a clean user experience.

## Installation

```bash
git clone https://github.com/AarjanAdhikari/monex.git

cd monex

npm install
```

## Development

```bash
npm run dev
```

Open http://localhost:3000

## Production

```bash
npm run build
npm run start
```

## Features

- Real-time currency conversion
- Live exchange rates
- Historical exchange rate charts
- Responsive interface
- Fast conversion
- Modern UI
- Dark mode

## Tech Stack

- Next.js
