# MONEX Currency Converter

MONEX is a premium, beautifully designed currency converter application built with Next.js, Tailwind CSS, and Recharts.

## Key Features
- Real-time exchange rates powered by the Frankfurter API.
- Historical data visualization with interactive charts (Recharts).
- Offline mode with cached data support.
- Fully responsive and accessible UI with a "Revolut/Apple" inspired premium design.
- Settings, history, favorites, and interactive legal definitions.
- Haptic feedback and smooth animations.

## Technologies Used
- Next.js (App Router)
- React
- Tailwind CSS
- Recharts
- Framer Motion
- Lucide React

## Highlights
This project demonstrates expertise in:
- High-quality, polished UI/UX implementation.
- Client-side data persistence and caching.
- Handling of real-time API integrations and data transformation.
- Advanced React patterns, custom hooks, and layout structuring.
